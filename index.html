🧠 The Core Loop

This isn't your typical "throw far = more money" game. Distance is an unlock mechanic, not a payout:

1. Charge your throw (nail the perfect window for +35% power)
2. Ball launches → physics simulation → lands at distance X
3. A Brainrot of that distance tier spawns at the landing spot
4. You become that Brainrot (yes, really)
5. The Giant Brainrot wakes up and chases you
6. RUN HOME (tap frantically to fill the escape bar before time runs out)
7. Made it? → Brainrot added to your base, generates coins/sec forever
Got caught? → You lose it; throw again
8. Passive income builds up → spend on throw upgrades + better balls
9. Throw farther → reach higher tiers → unlock $5000/sec Brainrots
10. REPEAT ∞

💰 Brainrot Tiers — Your Money Printers

Every throw spawns a Brainrot based on distance. Capture it, and it generates passive income every second:

Tier Distance Gate Income/Second Chase Difficulty
Common 0–100 m $1/s 🟢 Easy (most time)
Uncommon 100–300 m $3/s 🟢 Easy
Rare 300–600 m $10/s 🟡 Medium
Epic 600–1000 m $30/s 🟡 Medium
Legendary 1000–2000 m $100/s 🟠 Hard
Mythic 2000 m+ $500/s 🔴 Very Hard
SECRET Secret Zone $5000/s 💀 Brutal (least time)

Pro tip: Higher tiers = more money but scarier chases. That $5000/s Secret Brainrot? You'll have seconds to tap your way home before the Giant eats you.

🛠️ Tech Stack

Built entirely in Verse (Unreal Engine for Fortnite's scripting language) for UEFN (Unreal Editor for Fortnite). This is a proof-of-concept game you can play in Fortnite Creative.

Architecture Highlights

  • 🎮 9 modular systems (Physics, Throw, Brainrot, Base, Shop, Save, Distance, UI, GameManager)
  • 📐 Data-driven design — all balance numbers in one file (GameConstants.verse)
  • 🔒 Side-effect isolation — pure calculators separated from state mutation
  • 📦 No circular dependencies — clean downward dependency graph
  • 📚 Fully documented — README, Architecture Deep Dive, Technical Reference, Setup Guide

🎮 How to Play It

You'll need:

  • Unreal Editor for Fortnite (UEFN) — free download
  • An Epic Games account with Creator access

Then:

  1. Clone/download the GitHub repo
  2. Follow the Setup Guide to import the Verse code
  3. Place the game_manager device in your island
  4. Hit Play and start throwing!
Note: This is Fortnite Creative content—it runs inside Fortnite/UEFN, not in a web browser. Think of it like a Minecraft mod or Roblox game. This webpage is just to share the idea!
